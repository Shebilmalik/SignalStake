INSTALL OP_WALLET

OP_WALLET is the official Bitcoin wallet for OP_NET. It must be installed as a browser extension to connect.

1
Go to the official OP_WALLET releases page:
github.com/btc-vision/opwallet/releases →
2
Download the latest .zip file. Unzip it to a folder on your computer.
3
Open Chrome → chrome://extensions/
Enable Developer Mode → click Load unpacked
→ select the unzipped folder.
4
Reload this page. Then click CONNECT OP_WALLET.
